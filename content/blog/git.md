Git was created as a code version management tool.
<!-- more -->

## Usage

-   Rewind to the last commit but keep the file changes in the staged area.
    ```bash
    git reset HEAD~1
    ```
-   Untrack a file from Git but keep it.
    ```bash
    git rm --cached <FILE>
    ```
